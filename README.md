# Data Base
`Το πρόγραμμα αυτό υλοποιεί μία βάση δεδομένων για το ετήσιο φεστιβάλ του Pulse University.`

### Υποθέσεις:
- Το φεστιβάλ γίνεται μία φορά το χρόνο.
- Η τοποθεσία αλλάζει κάθε έτος.
- Οι τεχνικοί αναλαμβάνουν ανά εμφάνιση ενώ το υπόλοιπο προσωπικό ανά παράσταση
- Οι ερωτήσεις πρέπει να συμπεριλάβουν και τις προγραμματισμένες μελλοντικές παραστάσεις.
- *Στο query 9:* ως έτος θεωρήθηκε το έτος του φεστιβάλ.
- *Για το query 10:* το είδος μουσικής δεν είναι πεδίο του πίνακα artist αλλά σε βοηθητικό πίνακα π.χ. genres με σχέση one to many με τον πίνακα artist.
- *Για το query 12:* το τεχνικό προσωπικό ασφαλείας είναι το 5% του capacity, το βοηθητικό προσωπικό το 2%.
- *Για το query 15:* ο κάθε επισκέπτης βαθμολογεί συνολικά την μπάντα και έτσι αυτή συμμετέχει ως καλλιτέχνης.
- *Για το query 14:* Μερικές φορές το query 14 δεν έφερνε αποτελέσματα. Για τον λόγο αυτό έφτιαξα 2 βάσεις δεδομένων. Την odidb που δημιουργείται και διαγράφεται από τα install και load. Και την odisql που περιέχει συγκεκριμένα δεδομένα και εκεί εκτελούνται όλα τα queries.

### Σχόλια:
- Για τον σχολιασμό των index στα *query 4 και 6* έχει παραδωθεί αρχείο excel με αντιπαραβολή των αποτελεσμάτων, καθώς και αρχείο txt με τα σχόλια.
- Στο διάγραμμα Ε/R τα attributes παραλήφθηκαν για να γίνει πιο κατανοητό στον αναγνώστη. Μπορούν να βρεθούν στο relation diagram. 

## ▶️Usage
- Πρέπει η `php` να είναι σε `path`.
- Για να τρέξει πρέπει να εκτελεστεί το `startphp.bat`, που βρίσκεται στο `code`.
- Έπειτα ανοίγουμε έναν `browser` στη διεύθυνση https://localhost:5000

## ⌨Code
<ins>**Ο κώδικας περιέχει:**</ins>

- 📁Αρχείο: `install.sql`
    
    Δημιουργεί τη βάση δεδομένων με:
    - τους *πίνακες*
    - τα πρωτεύονται και δευτερεύονται *κλειδιά*
    - τους *triggers*, που εξασφαλίζουν την ορθότητα της βάσης

- 📁Αρχείο: `load.sql`

    Γεμίζει με *τυχαία δεδομένα* τη βάση.

- 📁Αρχεία τύπου: `QX.sql`

    Τα αρχεία αυτά *υλοποιούν* τα ερωτήματα μέσω *query*.

- 📁Αρχεία τύπου: `QX_out.txt`

    Αυτά περιέχουν *ενδεικτικά αποτελέσματα* εκτέλεσης των παραπάνω *query*.
